## Main Program Flow

```
┌─────────────────────────────────────────────────────────────────┐
│                        PROGRAM START                           │
└─────────────────────────────────────────────────────────────────┘
                                │
                                ▼
                     ┌─────────────────────┐
                     │   MAIN.ASM          │
                     │   Initialize        │
                     │   Data Segment      │
                     └─────────────────────┘
                                │
                                ▼
                     ┌─────────────────────┐
                     │   IO.ASM            │
                     │   Clear Screen      │
                     │   Set Video Mode    │
                     └─────────────────────┘
                                │
                                ▼
                     ┌─────────────────────┐
                     │   MENU LOOP         │
                     │   (Main Control)    │
                     └─────────────────────┘
                                │
                 ┌──────────────┼──────────────┐
                 ▼              ▼              ▼
         ┌─────────────┐ ┌─────────────┐ ┌─────────────┐
         │ Display Menu│ │ Get User    │ │ Process     │
         │ Options     │ │ Input       │ │ Selection   │
         └─────────────┘ └─────────────┘ └─────────────┘
                 │              │              │
                 └──────────────┼──────────────┘
                                ▼
                     ┌─────────────────────┐
                     │   ACTION ROUTER     │
                     └─────────────────────┘
                                │
            ╔═══════════════════╩═══════════════════╗
            ║                                       ║
    ┌───────▼───────┐                     ┌───────▼───────┐
    │ Option 1-4:   │                     │ Option 5:     │
    │ Draw Shapes   │                     │ Exit Program  │
    └───────┬───────┘                     └───────┬───────┘
            │                                     │
            ▼                                     ▼
    ┌─────────────┐                     ┌─────────────┐
    │ SHAPES.ASM  │                     │ MAIN.ASM    │
    │ Execute     │                     │ Terminate   │

    └─────────────┘                     └─────────────┘
            │                                     │
            ▼                                     ▼
    ┌─────────────┐                     ┌─────────────────┐
    │ Return to   │                     │ PROGRAM EXIT    │
    │ Menu Loop   │                     │ (Back to DOS)   │
    └─────────────┘                     └─────────────────┘
```
